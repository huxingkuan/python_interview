1. 安装软件

   ```linux
   sudo apt install 软件名
   ```

2. 卸载软件

   ```linux
   sudo apt remove 软件名
   ```

3. 更新可用软件包列表

   ```linux
   sudo apt update
   ```

4. 更新已安装的包

   ```linux
   sudo apt upgrade
   ```

5. 通常安装完ubuntu之后，可以先使用upgrade更新一下当前系统中可以升级的软件包

   ```linux
   sudo apt update
   sudo apt upgrade
   ```
